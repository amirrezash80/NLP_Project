# Ensemble Transformer for Cross-Lingual Semantic Textual Similarity

This repository implements and extends the paper:

**Piroozfar et al., 2024 – Ensemble Transformer for Cross-Lingual STS**

## Overview
We compute semantic similarity between Persian–English sentence pairs
without using machine translation, based on ensemble transformer models.
